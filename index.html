CSS3 Parallax Cover Section

This is a cool vertical parallax scrolling effect that can be created in pure CSS.

Note that the purpose of this pen is to act as a learning exercise, demonstrating the effects of CSS3 properties such as 3D transforms, SVG backgrounds, viewport units and so on.

At the moment mobile compatibility with this technique is a little scratchy and it works best in landscape orientation, so it's not ideal for production use.
